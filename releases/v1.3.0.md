# AWS Incident Response Playbook v1.3.0

**Release name:** Threat and Response Framework Alignment  
**Release date:** 2026-07-17

## Overview

Version 1.3.0 connects every scenario runbook to three complementary frameworks: MITRE ATT&CK Enterprise, NIST CSF 2.0 as profiled by NIST SP 800-61 Revision 3, and the AWS Well-Architected Security Pillar.

The mappings improve threat-language consistency, response-program traceability, and cloud-readiness assessment without changing the operational procedures introduced in earlier releases.

## Added

- Tailored framework-alignment table in all 20 runbooks.
- Central `docs/framework-mapping.md` cross-reference.
- MITRE ATT&CK technique context for each scenario.
- NIST CSF 2.0 function alignment.
- AWS Well-Architected SEC10 best-practice alignment.
- Repository-wide pull request template with technical, safety, and disclosure checks.

## Changed

- Updated README and documentation-index navigation.
- Updated the release badge to v1.3.0.
- Clarified that ATT&CK technique mappings are hypotheses or context until confirmed by evidence.

## Compatibility

- No breaking changes.
- Existing runbook procedures, commands, diagrams, and file paths remain compatible.

## Next release

Version 1.4.0 will improve decision support, domain navigation, and cross-runbook escalation paths.
