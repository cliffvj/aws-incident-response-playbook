# AWS Incident Response Playbook v2.0.0

**Release name:** Production Documentation  
**Release date:** 2026-07-17

## Overview

Version 2.0.0 completes Phase 2 and establishes a cohesive, production-inspired documentation foundation for the AWS Incident Response Playbook. The release integrates visual workflows, framework alignment, responder decision intelligence, domain navigation, authoritative references, and automated internal-link validation.

## Highlights

- Twenty scenario runbooks with incident snapshots, response flows, decision checkpoints, framework mappings, and navigation.
- Domain-oriented indexes for faster incident selection.
- Central MITRE ATT&CK, NIST, and AWS Well-Architected mapping guidance.
- Central authoritative-reference catalog.
- Mermaid source catalog and architecture-level response diagrams.
- Automated repository link validation through GitHub Actions.
- Complete release-history and roadmap integration.
