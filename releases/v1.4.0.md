# AWS Incident Response Playbook v1.4.0

**Release name:** Decision Intelligence

**Release date:** 2026-07-17

## Overview

Version 1.4.0 adds structured decision support to the AWS Incident Response Playbook. The release helps responders choose safer actions when severity, evidence, business impact, identity type, or recovery trust is uncertain.

## Added

- Tailored decision checkpoints in all 20 scenario runbooks.
- A central incident-response decision guide.
- Severity and escalation decision guidance.
- Evidence-before-action and containment-selection gates.
- Identity-specific containment paths.
- Trusted recovery and automation-approval gates.
- Closure criteria for operational and governance readiness.

## Changed

- Expanded README and documentation-index navigation.
- Added decision-support references to every scenario.
- Updated the pull request template with decision-safety validation.

## Compatibility

No breaking changes. Existing runbook procedures remain intact.

## Coming next

Version 2.0.0 will complete Phase 2 with repository-wide navigation, consistency validation, release consolidation, and production-documentation polish.
